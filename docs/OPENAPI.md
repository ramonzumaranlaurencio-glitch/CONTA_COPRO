# OpenAPI

FastAPI publica el contrato en:

- `GET /openapi.json`
- `GET /docs`

Para exportar un archivo versionable:

```bash
python scripts/export_openapi.py
```

El contrato incluye:

- Ledger y facturacion.
- Core financiero.
- Reportes.
- SUNAT Enterprise.
- IA.
- Integraciones.
- Health y metrics.
